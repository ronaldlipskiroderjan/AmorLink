# AmorLink

Bem-vindo ao repositório do AmorLink, nosso SaaS voltado para casais.

## Estrutura do Projeto
- `amor-api`: Backend em Java (Spring Boot)
- `amor-app`: Frontend em React/Vite (UI interativa)

## Como Rodar Localmente
Consulte os guias específicos dentro de cada pasta para instalar as dependências e iniciar os servidores locais.
